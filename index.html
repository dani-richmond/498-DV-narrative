Board Game Trends

Number of New Board Games Published per Year

Although board games have existed for several millennia, the number created per year remained relatively low until the 1960's. The number of new games created each year slowly grew at first but has really exploded in recent years although it has trended downward the last two years.
The board game boom started around 2000 and the more recent explosion after 2010 with a resurgence in gaming stores and cafes. The data displayed here and on the following graphs is from BoardGameGeek, the go-to website for all things board game related.
